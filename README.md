# KIT Mentoring Keyword Roulette

KIT 4기 멘토링 운영을 위한 정적 웹 기반 키워드 룰렛입니다.

멘토링 모둠 활동에서 키워드 박스를 선택하고 룰렛을 돌려 대화 주제를 뽑을 수 있습니다. 발표나 교실 운영 상황에 맞춰 전체 화면, 타이머, 카테고리 전환을 지원합니다.

## 구성

- `index.html`: 앱 화면 구조
- `styles.css`: 레이아웃과 시각 스타일
- `app.js`: 룰렛, 카테고리, 타이머 동작
- `assets/`: 배경 이미지와 Paperlogy 폰트

## 실행 방법

별도 설치 없이 `index.html` 파일을 브라우저에서 열면 실행됩니다.

로컬 서버로 확인하려면 다음 명령을 사용할 수 있습니다.

```bash
python -m http.server 8080
```

그다음 브라우저에서 `http://localhost:8080`으로 접속합니다.

## 사용 방법

1. 상단에서 키워드 박스를 선택합니다.
2. 가운데 `START` 버튼을 눌러 룰렛을 돌립니다.
3. 오른쪽 타이머 시간을 조정하고 `시작` 버튼을 눌러 모둠 대화 시간을 관리합니다.
4. 우측 상단 버튼으로 전체 화면을 켤 수 있습니다.

스페이스바를 눌러도 룰렛을 돌릴 수 있습니다.

## 키워드 박스

- 스몰토크
- 학교
- 공부
- 마음
- 진로
- 전체

## 배포

GitHub Pages에서 정적 사이트로 바로 배포할 수 있습니다.

Repository Settings → Pages → Deploy from a branch를 선택한 뒤 `main` 브랜치의 `/ (root)`를 지정하면 됩니다.
